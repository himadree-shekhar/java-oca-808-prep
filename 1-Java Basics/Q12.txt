Question: 
Which of the following are features of Java?
Some candidates have reported a similar question being asked with a slightly different (and ambiguous) wording: 
Which of the following are objected oriented features of Java?

You had to select 2 options :
(A) Every class must have a main method so that it can be tested individually from command line.
(B) Every class belongs to a package.
(C) A package must have more than one class.
(D) A class may inherit from another class.

-------------------------------------------------------------------
Answer : (B),(D)
(B) Why : in Java, every class belongs to a package, even if not explicitly declared (unnamed package)
          The package in Java is considered as object-oriented features because it gives : Encapsulation, Access Control, Reusability etc.
 
(D) Why : java inheritance
