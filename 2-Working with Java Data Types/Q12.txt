Q12
Will The following code snippet will print 4 ?

int i1 = 1, i2 = 2, i3 = 3; // multiple variable declaration : OK 
int i4 = i1 + (i2=i3 );
System.out.println(i4);


Please select 1 options :

(A) TRUE
(B) FALSE
-------------------------------------------------------------------
Answer : 
(A) TRUE

NOTE: 
int i4 = i1 + (i2=i3 ); // left-side of assignment(=) must be variable. Parentheses Ensure it
